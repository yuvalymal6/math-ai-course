## 0. נושאי הבגרות — שאלון 804

| תחום | נושאים |
|------|--------|
| חד"א (חשבון דיפרנציאלי ואינטגרלי) | פולינומים (נגזרת, חקירה), פונקציות רציונליות, פונקציות שורש |
| גיאומטריה אנליטית | משוואת ישר, מעגל, משיק למעגל |
| קיצון | בעיות קיצון גיאומטריות, קיצון בפונקציות |
| הסתברות | עץ הסתברות, טבלה דו-ממדית, הסתברות מותנית |
| סטטיסטיקה | התפלגות נורמלית, ממוצע משוקלל |

### הדגשים מרכזיים
- **חקירת פונקציות** היא לב הבגרות — מונוטוניות, קיצוניים, נגזרת, סקיצה
- **הסתברות מותנית** — P(A|B) = P(A∩B)/P(B)
- **קיצון** — בעיות מילוליות שדורשות בניית פונקציה, גזירה, ובדיקה

## 1. מבנה כללי — הדפוס החוזר

כל דף תרגיל ברמה 100 מכיל את **כל** הרכיבים הבאים:

### 1.1 שלוש רמות קושי
| רמה | תווית | צבע | קומפוננטה | רמת קושי בגרות |
|------|--------|------|-----------|-----------------|
| מתחיל | basic | ירוק `#16A34A` | `LadderBase` — העתקת פרומפטים + דיווח עצמי | נגיש — שאלה ישירה |
| בינוני | medium | כתום `#EA580C` | `LadderMedium` — כתיבת פרומפט + ולידציה (ניקוד 75+) | ~50-60% קושי בגרות |
| מתקדם | advanced | אדום `#DC2626` | `LadderAdvanced` — `MasterPromptGate` + ניקוד 90+ | ~95% קושי בגרות |

### 1.2 רכיבים חובה לכל רמה
- **דיאגרמת SVG שקטה** — ללא מספרים, ללא תשובות. רק צורות, תוויות, צבעי תפקיד
- **טקסט השאלה** — מנוסח בעברית, מחולק לסעיפים (א׳/ב׳/ג׳/ד׳)
- **מוקשים נפוצים (Pitfalls)** — 2-3 כרטיסי אזהרה (מצביעים על טעות, לא חושפים פתרון)
- **פרומפט זהב (Golden Prompt)** — פרומפט ראשי מלא להעתקה (חוץ מרמה מתקדמת)
- **סולם פרומפטים (Ladder)** — שלבים עם פרומפטים לכל סעיף
- **מעבדה אינטראקטיבית (Lab)** — סליידרים + SVG דינמי + אריחי נתונים

### 1.3 רכיבי עטיפה
- **`SubtopicProgress`** — סרגל התקדמות כללי
- **`MarkComplete`** — כפתור סיום
- **`LabMessage`** — הודעות מותנות במעבדה
- **כפתור חזרה** — לינק לדף האב

## 2. הפרומפט הזהב — מבנה אחיד

כל פרומפט זהב עוקב אחרי פרוטוקול 3 חלקים:

```
1. הגדרת תפקיד: "אתה המורה הפרטי שלי / אני תלמיד כיתה יא׳"
2. מגבלה: "אל תפתור עבורי — שאל אותי שאלות מכווינות"
3. שיטת עבודה: "סרוק את הנתונים בלבד. תעצור אחרי כל שלב ותחכה שאגיד להמשיך"
```

**אורך אופייני:** 150-400 תווים. פרומפטי שלבים: 50-120 תווים (משפט אחד ממוקד, כולל נתוני השאלה).

---

## 3. פרומפטי שלבים — סגנון הניסוח

### ציווי ישיר בגוף שני
- "תדריך אותי איך..." / "תסביר לי..." / "הנחה אותי..."

### עצמאיים — כוללים נתוני השאלה
- "נתונה הפונקציה f(x) = x³ - 3x² + 4. תנחה אותי למצוא את נקודות הקיצון."

### מגבלות מפורשות
- "אל תפתור" / "אל תיתן תשובה סופית"

## 4. דיאגרמות SVG שקטות — כללים

| כלל | פירוט |
|------|--------|
| אין מספרים | לא `6`, לא `82`, לא `√52` |
| אין תשובות | לא `שטח = 48`, לא `x = 7` |
| תוויות בלבד | A, B, C, f, g, ℓ₁ |
| צבעי תפקיד | ענבר `#f59e0b` (בסיסי), סגול `#a78bfa` (מתקדם), אמרלד `#34d399` (פתרון), צפחה `#64748b` (עזר) |
| קווים מוסתרים | `strokeDasharray="4,3"`, צבע `#334155` |
| גודל | `className="w-full max-w-sm mx-auto"`, `viewBox` בלבד |

### דיאגרמות ספציפיות לכיתה יא
- **חקירת פונקציה:** עקומה כללית עם סימוני קיצון (? על נקודות), ציר x מסומן, ללא ערכים
- **מעגל:** מעגל עם מרכז O ורדיוס מקווקו, נקודת משיק מסומנת, ללא קואורדינטות
- **עץ הסתברות:** ענפים עם תוויות אירועים (A, B, Ā, B̄), ללא ערכי הסתברות

## 5. מעבדות אינטראקטיביות — דפוס

### מבנה אחיד
1. **סליידרים** — `<input type="range">` לכל פרמטר
2. **חישוב חי** — ללא כפתור submit
3. **SVG דינמי** — משתנה בכל תזוזת סליידר
4. **אריחי נתונים** — `grid grid-cols-N gap-3`, כל אריח: תווית + ערך
5. **תווית תחתונה** — פרשנות מתמטית

### דוגמאות למעבדות כיתה יא
| נושא | מעבדה | סליידרים | אריחים |
|------|--------|----------|--------|
| חקירת פולינום | `DerivativeLab` | 3 (a, b, c) | f'(x), נק׳ קיצון, מונוטוניות |
| מעגל ומשיק | `CircleLab` | 3 (cx, cy, r) | משוואה, משיק, מרחק |
| הסתברות | `ProbabilityLab` | 2 (P(A), P(B\|A)) | P(A∩B), P(B), עץ |
| קיצון גיאומטרי | `OptimizationLab` | 1 (x) | שטח, היקף, מקסימום |

### עיצוב — Island Protocol
```tsx
style={{ border: "8px solid #334155", borderRadius: "40px", padding: "2.5rem", background: "#020617" }}
```

## 9. שגיאות נפוצות (Pitfalls) — כללים

- **מצביע על טעות, לא חושף פתרון**
- ✅ טוב: "תלמידים שוכחים לבדוק f''(x) — בלי זה לא ניתן לקבוע אם זה מקסימום או מינימום"
- ❌ רע: "הציבו f'(x)=0 ופתרו x=2, x=-1" — זה הפתרון עצמו
- 2 שגיאות לכל רמה
